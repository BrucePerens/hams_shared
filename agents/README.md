# HAMS Shared Agents

This directory contains the custom Antigravity agent skills, workflows, and rules used across the `hams_open` project.

## Purpose
These agents and skills provide automated assistance for tasks such as codebase reviews, documentation generation, Odoo development, linter compliance, and testing. They define the specific behavioral guidelines and orchestration mechanisms (like the divide-and-conquer architecture) necessary for contributing to the project.

## Usage
The skills defined in this directory are automatically loaded by the Antigravity agent when the workspace is properly configured. They provide context, guidelines, and tools that the agent can trigger when performing specific tasks.
